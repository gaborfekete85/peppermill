## What was verified

`python -m peppermill` runs end-to-end against the real Anthropic API:

1. AnthropicProvider constructed (ANTHROPIC_API_KEY env var picked up).
2. CLI channel reads stdin.
3. AgentLoop publishes user message → calls AnthropicProvider.chat().
4. POST /v1/messages to api.anthropic.com hits Claude Haiku 4.5.
5. Response parsed; tool_use block (if any) executes AddTool.
6. Final text response published to stdout.
