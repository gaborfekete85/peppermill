### Known Limitations (Deferred)
- **No atomic writes:** Simple append strategy. If process crashes mid-write, a partial JSON line may corrupt file. Upgrade in v0.9.
- **No TTL/cleanup:** Old sessions accumulate forever. Add cleanup in v0.5.
- **No history format versioning:** If TurnSummary changes, old files may break. Add versioning in v0.5.
- **No per-user sessions:** All users in a chat_id share history. Add per-user isolation in v0.10.
- **Graceful error handling:** If save fails, we log and continue. No retries. Upgrade in v0.9.
