# PepperMill v0.3 Implementation Plan — TurnState FSM + AgentRunner extraction

> **Execution mode for this version:** Claude executes the tasks directly with one commit per task. The plan is still the long-term record. User reviews via commits, not copy-paste.

**Goal:** Promote `AgentLoop` to a small `TurnState` FSM (`BUILD → RUN → RESPOND → DONE`) routed via `match/case`, and move the LLM-conversation engine (provider call + tool execution) into a new `AgentRunner` class with `AgentRunSpec` / `AgentRunResult` dataclasses.

**Architecture:** Two new units. `AgentRunner` is pure LLM conversation — given a spec, it loops over `provider.chat()` and tool execution until a final text reply or `max_iterations`. `AgentLoop` becomes pure per-turn orchestration — it builds a spec for the runner, calls it, publishes the result as an `OutboundMessage`. Each state of the FSM is a method that takes a `_TurnContext` and returns the next state. Behaviour from v0.2 is preserved end-to-end.

**Tech Stack:** Python 3.14, `enum.Enum` + `auto()`, `match/case`, `dataclasses`, pytest, ruff.

**Reference:** v0.3 design spec at [`docs/superpowers/specs/2026-06-03-v0.3-fsm-and-runner-design.md`](../specs/2026-06-03-v0.3-fsm-and-runner-design.md). Project roadmap at [`docs/superpowers/specs/2026-05-29-peppermill-reimplementation-design.md`](../specs/2026-05-29-peppermill-reimplementation-design.md).

**Branch:** Work happens on `v0.3` (already created from `main` after v0.2 merge).

## Task 5: ScriptedRunner test helper

**Files:** Create `tests/_helpers/scripted_runner.py`.

```python
"""ScriptedRunner — test-only stand-in for AgentRunner used by AgentLoop tests."""
from __future__ import annotations

from peppermill.agent.runner import AgentRunResult, AgentRunSpec


class ScriptedRunner:
    def __init__(self, results: list[AgentRunResult]) -> None:
        self._results = list(results)
        self._idx = 0
        self.last_spec: AgentRunSpec | None = None

    async def run(self, spec: AgentRunSpec) -> AgentRunResult:
        self.last_spec = spec
        if self._idx >= len(self._results):
            raise RuntimeError("ScriptedRunner exhausted")
        r = self._results[self._idx]
        self._idx += 1
        return r
```

Tiny smoke test in same task or rolled into Task 6's loop tests. Commit with `test(v0.3): ScriptedRunner helper for hermetic AgentLoop tests`.

import pytest

from peppermill.agent.loop import AgentLoop, TurnState
from peppermill.agent.runner import AgentRunResult
from peppermill.agent.tools.add import AddTool
from peppermill.bus.events import InboundMessage
from peppermill.bus.queue import MessageBus

from tests._helpers.scripted_provider import ScriptedProvider  # used to satisfy AgentRunner construction
from tests._helpers.scripted_runner import ScriptedRunner
from peppermill.providers.base import LLMResponse


@pytest.fixture
def bus():
    return MessageBus()


async def test_run_once_publishes_outbound_with_runner_result(bus):
    runner = ScriptedRunner([
        AgentRunResult(
            final_content="hello back",
            messages=[],
            tools_used=[],
            stop_reason="completed",
        ),
    ])
    loop = AgentLoop(
        bus=bus,
        provider=ScriptedProvider([LLMResponse(content="x")]),  # ignored — runner is scripted
        tools={"add": AddTool()},
    )
    loop._runner = runner  # inject

    await loop.run_once(InboundMessage(channel="cli", sender_id="u", chat_id="c", content="hi"))

    out = await bus.consume_outbound()
    assert out.content == "hello back"
    assert out.channel == "cli"
    assert out.chat_id == "c"


async def test_run_once_publishes_empty_string_when_runner_returns_none_content(bus):

        AgentRunResult(final_content=None, messages=[], tools_used=[], stop_reason="max_iterations"),

    loop = AgentLoop(
        bus=bus,
        provider=ScriptedProvider([LLMResponse(content="x")]),
        tools={},
    )
    loop._runner = runner
