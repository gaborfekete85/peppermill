# PepperMill

A minimal LLM-agent framework, reimplemented for deep understanding.

This project is a learning reimplementation of [nanobot](../nanobot/), built
step by step on long-lived branches (`v0.1`, `v0.2`, …). See the project
roadmap: [`docs/superpowers/specs/2026-05-29-peppermill-reimplementation-design.md`](docs/superpowers/specs/2026-05-29-peppermill-reimplementation-design.md).
